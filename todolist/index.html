To-do list

In this simple to-do list, use the text input below to add an item to the list. Items in the list can then be marked as done with the checkbox; can be unmarked if not actually done; and removed from the list by clicking "Remove" on its right. You can clear the list at any time by pressing the "Clear list" button above the to-do list.
