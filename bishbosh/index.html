Bish-Bosh 2.0: The Bish-Bosher

Description

This page lets you specify three parameters:

Bish
Number to replace with 'Bish' in the output.
Bosh
Number to replace with 'Bosh' in the output.
Max
The maximum number to print up to.

Then it will go through all values between 1 and Max and print them out. However, every value divisible by only Bish will be replaced with 'Bish'; those divisible by only Bosh are replaced with 'Bosh'; and those divisible by both are replaced with 'Bish-Bosh'.
